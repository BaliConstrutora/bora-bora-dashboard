## Objetivo
Renomear o projeto Lovable para o nome escolhido pelo usuário: **Bora Bora - Gestão Comercial**.

## Observação importante
O nome do projeto no Lovable é apenas um rótulo de organização. Ele **não altera**:
- URLs publicadas/preview
- Código-fonte
- Banco de dados
- Configurações de autenticação

O código do app já está alinhado: o título no `<head>` e na sidebar já usam "Bora Bora" / "Bora Bora - Gestão Comercial".

## Passos
1. No editor do Lovable, clicar no **nome atual do projeto** no topo esquerdo.
2. Selecionar a opção **"Rename project"**.
3. Digitar o novo nome: `Bora Bora - Gestão Comercial`.
4. Confirmar.

## Verificação
- O nome no topo esquerdo deve passar a exibir "Bora Bora - Gestão Comercial".
- Nenhuma alteração de código é necessária, pois as referências internas já estão corretas.

## Nota
Se desejar também ajustar algum texto visível no app (ex: subtítulo, metadados OG, fallback do header), podemos fazer isso em uma etapa separada.
